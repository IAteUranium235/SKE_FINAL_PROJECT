# Tower Defence 3D

## Project Description

- Game Genre: Tower Defence, Third-Person Action

Tower Defence 3D is a game that combines tower defence strategy with a third-person 3D perspective, built entirely with Python and Pygame using a custom software renderer (no OpenGL).

## Installation

To clone this project:

```sh
git clone https://github.com/IAteUranium235/SKE_FINAL_PROJECT.git
cd SKE_FINAL_PROJECT
```

### Windows

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Mac

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Tutorial / Usage

| Key | Action |
|-----|--------|
| W / A / S / D | Move player |
| Mouse | Look around |
| E | Open / close shop |
| 1 | Equip Wrench |
| Left Click (Wrench) | Place / remove tower |
| ESC | Pause menu |
| TAB | Toggle mouse lock |

Walk to the **shop** (behind the grid) and press **E** to buy towers. Use the **Wrench** to place towers on the grid. Defend the base from enemy waves.

---

## Game Features

- Custom 3D software renderer using Painter's Algorithm
- 10 levels including 2 boss levels (level 5 and 10)
- 9 tower types with unlock progression by level
- 8+ enemy types with unique abilities (teleport, spawn minions, etc.)
- Boss system with phase-based difficulty and enemy summoning
- Passive income + wave clear bonus economy
- Full audio system with wave-based music transitions
- Statistical data recording with CSV export and graph visualization
